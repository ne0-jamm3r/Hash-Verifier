
# Hash Verifier

**With this application, you can easily verify hashes without the need to download anything to the internet and without dealing with the command line.**

<img src="photo/logo.png" width="200">

## Supported Algorithms:

1. _MD5_
2. _SHA1_
3. _SHA256_
3. _SHA384_
5. _SHA512_
